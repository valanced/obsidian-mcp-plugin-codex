# Codex Notes

The main fork documentation now lives in [README.md](README.md).

This file is kept for older links. It describes the same Codex-focused fork of
[aaronsb/obsidian-mcp-plugin](https://github.com/aaronsb/obsidian-mcp-plugin):
an Obsidian plugin that exposes a local MCP server, plus a Codex marketplace
package that starts a configurable HTTP/HTTPS bridge.

Use these core variables when configuring Codex:

```bash
export OBSIDIAN_MCP_URL="http://localhost:3001/mcp"
export OBSIDIAN_MCP_API_KEY="paste-api-key-from-obsidian-settings"
```

For HTTPS, use `https://localhost:3443/mcp` and set `NODE_EXTRA_CA_CERTS` to the
certificate generated by the Obsidian plugin.
